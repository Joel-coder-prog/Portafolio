Herramientas


Utilizando programas desde mi pc realizo diferentes tareas gracias a:

Visual Studio Code

Es un editor de código fuente, compatible con varios lenguajes de programación y frameworks (librerias). En mi caso utlizo HTML donde realizo el armado de estructura básica de la pagina usando texto, imágenes, iconos, enlaces, etc. CSS donde armo el diseño y creo estilos a la estructura del codigo y JAVASCRIPT donde creo funciones animadas a la pagina web.

Github

Es considerado como un dominio online gratuito donde subo mis proyectos y modifico el codigo si es requerido. Es muy usado por los desarrolladores para subir sus archivos como web.

Canvas

Es una pagina web usada como herramienta para analizar y crear modelos de negocio de forma simplificada (curriculums, videos publicitarios, logos, portadas, targetas, etc.). Mayormente en mi caso lo utilizo para diseño de logos o portadas.

Photoshop

Permite modificar imágenes y fotografías. Es un programa reconocido por su uso en la creación y edición de imágenes como gráficos o logos de marcas, mayormente lo utilizo para recortes de imagenes para transformarlos en objetos.

¿Por qué deberías tener un sitio web?

En la actualidad, tener un sitio web propio se ha convertido en una necesidad fundamental para cualquier persona que quiera establecer una presencia sólida en línea. Ya sea que seas un profesional independiente, un artista, un emprendedor o simplemente alguien que quiere compartir su pasión o conocimientos con el mundo, un sitio web te brinda innumerables beneficios.

Un sitio web te brinda la oportunidad de expandir tu alcance y llegar a una audiencia más amplia. Con tu presencia en línea, puedes llegar a personas de todo el mundo, ampliando tus oportunidades de networking, colaboración y crecimiento profesional.

La necesidad de tener un sitio web

tener tu propio sitio web te da control total sobre tu imagen en línea. Puedes diseñar y personalizar tu sitio de acuerdo a tus preferencias y estilo, creando una identidad digital única que te represente. También puedes optimizar tu sitio web para mejorar su visibilidad en los motores de búsqueda, lo que te ayudará a destacar en línea y ser encontrado por potenciales clientes o empleadores.

No importa cuál sea tu profesión o pasión, tener tu propio sitio web te brinda la oportunidad de destacar en el mundo digital y aprovechar al máximo las oportunidades que ofrece Internet.
